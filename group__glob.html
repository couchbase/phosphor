Global Events

Macros

#define TRACE_GLOBAL0(category, name)
 

Detailed Description

Global events are used for events that are not scoped to a thread and do not conceptually have a duration. Examples of this might include the initiation of a system shutdown.

Example: 

TRACE_GLOBAL0("Memcached", "Shutdown")
